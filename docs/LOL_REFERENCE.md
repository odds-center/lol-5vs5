# 리그 오브 레전드(LoL) 내전 사이트 구축을 위한 핵심 정보

내전(커스텀 게임) 팀 밸런싱 사이트 개발 시 참고할 롤의 핵심 데이터와 고려사항을 정리한 문서입니다.

---

## 1. 티어 및 랭크 시스템 (팀 밸런싱의 핵심)

유저 실력을 수치화하기 위한 데이터. **에메랄드** 티어 포함.

### 일반 티어 (4단계 세분화: 예시 골드 4 ~ 골드 1)

| 티어 | 영문 |
|------|------|
| 아이언 | Iron |
| 브론즈 | Bronze |
| 실버 | Silver |
| 골드 | Gold |
| 플래티넘 | Platinum |
| 에메랄드 | Emerald |
| 다이아몬드 | Diamond |

### 천상계 티어 (단계 없음, LP로 구분)

| 티어 | 영문 |
|------|------|
| 마스터 | Master |
| 그랜드마스터 | Grandmaster |
| 챌린저 | Challenger |

### 개발 팁: 티어 점수화

내전 밸런스 시 티어를 점수로 환산하는 로직이 필요합니다.

- 예: **아이언 4 = 1점** 기준, 티어·단계가 올라갈 때마다 점수 가산  
  (브론즈 4 = 5점, 실버 4 = 9점 …)
- 현재 본 프로젝트는 **MMR(숫자)** 을 사용자가 직접 입력하는 방식으로, 위 티어→점수 환산은 추후 확장 시 적용 가능

---

## 2. 포지션 (역할군)

5대5 게임이며, 각 구역·역할이 명확합니다. 신청 시 **주 포지션 / 부 포지션** 선택을 고려할 수 있습니다.

| 포지션 | 영문 | 설명 |
|--------|------|------|
| 탑 | Top | 상단 공격로. 탱커·브루저(전사) |
| 정글 | Jungle | 맵 몬스터 사냥, 라인 지원. 게임 흐름·밸런스 영향 큼 |
| 미드 | Mid | 중단 공격로. 마법사·암살자, 캐리력 높음 |
| 원딜 | ADC / Bottom | 하단 공격로 핵심 딜러, 후반 캐리 |
| 서폿 | Support | 하단 원딜 보좌, 시야 담당 |

---

## 3. 내전 밸런싱 시스템 기획 시 고려사항

단순 티어 합산만으로는 밸런스가 붕괴될 수 있음. 아래 요소를 로직에 반영하는 것을 권장합니다.

### 주 포지션 / 부 포지션 페널티

- 다이아 유저가 **주 포지션(미드)** 을 가면 100% 실력, **비주류 포지션(정글)** 이면 플래티넘 수준이 될 수 있음
- **비주류 포지션 배정 시 티어(또는 MMR) 점수 차감** 로직 적용 권장

### 핵심 포지션 격차 제한

- **정글·미드** 실력 차이가 양 팀 간 크면 게임이 일방적으로 끝남
- 양 팀의 **정글+미드 티어(또는 MMR) 합**을 비슷하게 맞추는 **가중치 시스템** 고려

### 듀오 신청 기능

- 바텀(원딜+서폿) 듀오, 미드–정글 듀오 등 **같이 팀에 묶어 주는 기능** 시 유저 만족도 상승
- 추후 기능 확장 시 도입 검토

---

## 4. 연동 시 고려: 외부 데이터 (Riot API)

유저가 직접 티어/MMR을 입력하면 거짓 입력(주작) 가능성이 있습니다.

- **Riot Games API**: **소환사명(닉네임) + 태그** 입력 시, 공식 API로 **실시간 솔로 랭크 티어**·**모스트 챔피언** 등을 조회해 자동 반영
- 현재 본 프로젝트는 **클라이언트 전용(localStorage)** 이므로 API 연동 시 서버/프록시 또는 CORS 정책 확인 필요

## 5. 본 프로젝트(LoL 5vs5 내전)에서의 적용 현황

| 항목 | 현재 구현 |
|------|-----------|
| 실력 수치 | MMR 사용자 입력 (티어→점수 자동 환산 미구현) |
| 포지션 | 탑/정글/미드/원딜/서폿 선택(선호·포지션 금지) |
| 팀 나누기 | MMR 스네이크 드래프트(평균 밸런스), 다시 나누기 시 랜덤 |
| 포지션 페널티 | 미적용 (동일 MMR 내 역할 랜덤) |
| 정글/미드 가중치 | 미적용 |
| 듀오 | 미적용 |
| Riot API | 미연동 |

추가 기획 시 위 표와 본 문서를 기준으로 확장하면 됩니다.
